# Substation Architect

Production-ready Angular project for the Substation single-line diagram editor.

## Requirements
- Node.js 18+ (recommended 20 LTS)
- npm 9+

## Setup
```bash
npm install
```

## Development
```bash
npm run start
```

## Production Build
```bash
npm run build:prod
```

## Lint and Test
```bash
npm run lint
npm run test
```

## Notes
- Tailwind CSS is configured via `tailwind.config.js` and `postcss.config.js`.
- The app is a standalone Angular component bootstrapped in `src/main.ts`.
